# 🎵 Spotify Music Trends Analyzer

A Data Science + ML web app that analyses 114,000 Spotify songs!

## 🔗 Live App
https://spotify-music-trends.streamlit.app

## 📊 Features
- Top 10 Most Popular Songs
- Genre vs Music Features Analysis
- Mood (Valence) by Genre
- 🤖 ML Song Popularity Predictor
- 🔍 Song Search with Radar Chart

## 🛠️ Tech Stack
Python | Pandas | Matplotlib | Seaborn | Streamlit | Scikit-learn

## 📁 Dataset
Spotify Tracks Dataset — 114,000 songs from Kaggle

## 🚀 Run Locally
pip install -r app/requirements.txt
streamlit run app/dashboard.py
